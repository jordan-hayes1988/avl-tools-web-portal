## Overview

AVL Tools Portal provides a browser-ready HTML interface that acts as a single entry point for opening tools from one location. It is a good fit for projects that want a web page front end instead of a separate desktop app.

The portal is organized so users can get to the available tools quickly, without extra setup steps or complex navigation. As a web-first repository, it offers a simple, lightweight way to group tool access into one easy-to-browse place.

## Usage

Launch the portal in a browser and pick the tool entry you want to open.

Typical workflow:

1. Deploy or open the HTML files.
2. Load the portal homepage.
3. Choose a tool from the interface.
4. Follow the linked destination or embedded navigation.

If you are hosting it locally, refresh the page after making content changes so the latest HTML updates are visible.
